# YouTube Channel Viewer

YouTubeチャンネルの動画一覧を表示し、動画の再生とコメント表示ができるWebアプリケーションです。

## 主な機能

- チャンネル検索
- 動画一覧表示
- 動画再生
- コメント表示
- レスポンシブデザイン

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- YouTube Data API v3
